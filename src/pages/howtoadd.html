How to add Jupyter on Trust Wallet

Step 1. Search for the Token

Tap on the toggle sign in the upper right corner of the main Wallet screen. Search for the token, and if the token is not available, you will see an Add Custom Token button.

Step 2. Fill in Token Details

Tap on Add Custom Token button, then make sure that you have selected the correct Network. For this guide, we need to select Smart Chain on the supported Network, since this is a token that resides on the Binance Smart Chain. Click Ethereum

Click Smart Chain

After pasting the Contract Address, the other details like the Name, Symbol, and Decimals will be automatically filled up. If not, fill them in manually. Tap on Save to continue. Contract Adress is: 0xD6ea4ec77fB7e0d5c331F9109f00CcEdE255b9C8

Step 4. Token Successfully Added

You have successfully added the Jupyer $IOM Token.
